# win10-ubuntu 使用总结笔记 #
win10 子系统功能： Linux 安装了 Ubuntu 子系统。

## 设置步骤 ##
win10 打开控制面板->程序->程序或功能->启用或关闭window功能->适用于Linux的Windows子系统(勾选这一项)->提示重启电脑(重启)->win10应用商店搜索->选择安装Linux(个人安装Ubuntu)->等待->启用和设置密码->可在cmd:bash开启。
