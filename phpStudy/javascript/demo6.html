javascript设计模式之封装(一)

在JavaScript王国--多态案例中我们了解了JavaScript的多态性，接着我们去学习了解封装。

在许多语言的对象系统中，封装数据是由语法解析来实现的，这些语言也许提供了 private、public、 protected 等关键字来提供不同的访问权限。但JavaScript并没有提供对这些关键字的支持，我们只能依赖变量的作用域来实现封装特性，而且javascript只能模拟出private和public两种封装性。

				<script type="text/javascript">
					//封装函数
					var myObject = (function(){
						var __name = 'sven';//私有(private)变量
						return {
							getName:function(){//公开(public)方法
								return __name;
							}
						}
					})();
					console.log( myObject.getName() ); // 输出： sven
					console.log( myObject.__name ) // 输出： undefined
				</script>

具体效果，请单击右键查看源代码->选择控制台，就可以观察具体效果。
